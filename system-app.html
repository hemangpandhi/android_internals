System App

Core apps with elevated privileges

What are System Apps?

System apps are pre-installed applications that reside in the system partition of an Android device. They have elevated privileges and are essential for core device functionality.

Privileges of System Apps

  • Access to protected system APIs
  • Cannot be uninstalled by regular users
  • Can request special permissions not available to user apps

Differences from User Apps

  • User apps are installed in the data partition and can be removed by users.
  • System apps are updated via OTA or firmware updates.
  • System apps can act as device administrators.

Examples

  • Settings
  • Phone
  • System UI
  • Package Installer
