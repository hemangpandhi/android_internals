Android Commands

Essential shell and system commands for Android

Common Android Shell Commands

  • pm list packages - List all installed packages
  • am start -n <package/activity> - Start an activity
  • settings get <namespace> <key> - Get system setting
  • logcat - View system logs
  • top - Show running processes
  • df - Show disk usage
  • getprop - Get system properties
  • svc wifi enable/disable - Control Wi-Fi

Usage Examples

  • adb shell pm list packages | grep google - List Google packages
  • adb shell am start -a android.intent.action.VIEW -d https://www.android.com - Open URL in browser
